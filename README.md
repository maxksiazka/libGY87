# GY-87 embedded driver with Hardware Abstraction Layer
---
A high performance embedded driver for the GY-87 sensor module, which includes an accelerometer, gyroscope, and magnetometer. This driver is designed to provide a Hardware Abstraction Layer (HAL) for easy integration into various embedded systems.

## 📜 Lineage & Credits

This project is a fork of the original GY-87 driver developed by **rizacelik**. The original driver can be found at [Original Repository Link](https://github.com/rizacelik/GY87). 
I attempt to make significant improvements and optimizations to enhance performance and compatibility with a wider range of microcontrollers.

## 🚀 Features
- Support for accelerometer, gyroscope, and magnetometer data acquisition.
- Hardware Abstraction Layer (HAL) for easy integration with different microcontroller platforms.
